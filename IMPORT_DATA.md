# Импорт данных из Excel

После инициализации базы данных можно импортировать данные из Excel файлов, расположенных в папке `pril`:

## Файлы для импорта:

1. **user_import.xlsx** - пользователи системы
2. **Tovar.xlsx** - товары
3. **Заказ_import.xlsx** - заказы
4. **Пункты выдачи_import.xlsx** - пункты выдачи (опционально)

## Запуск импорта:

```bash
python migrations\import_excel.py
```

Скрипт автоматически:
- Импортирует пользователей из `user_import.xlsx`
- Импортирует товары из `Tovar.xlsx` (включая изображения из папки pril)
- Импортирует заказы из `Заказ_import.xlsx`

## Формат данных:

### Пользователи (user_import.xlsx):
- Логин
- Пароль
- ФИО
- Роль (admin, manager, client)

### Товары (Tovar.xlsx):
- Наименование
- Категория
- Описание
- Производитель
- Поставщик
- Цена
- Единица измерения
- Количество на складе
- Скидка
- Фото (номер файла из папки pril, например: 1.jpg)

### Заказы (Заказ_import.xlsx):
- Артикул
- Статус
- Адрес пункта выдачи
- Дата заказа
- Дата доставки
