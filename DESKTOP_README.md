# Десктопная версия приложения ООО «Обувь»

## Описание

Десктопная версия системы управления продажей обуви, созданная на базе Flet. Приложение использует ту же базу данных и бизнес-логику, что и веб-версия.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Убедитесь, что база данных `shoe_store.db` существует (или создайте её через миграции)

## Запуск

Для запуска десктопного приложения выполните:

```bash
python desktop_app.py
```

## Функциональность

- **Авторизация**: Вход в систему с использованием логина и пароля
- **Управление товарами**: 
  - Просмотр списка товаров
  - Поиск и фильтрация (для менеджеров и администраторов)
  - Добавление, редактирование и удаление товаров (для администраторов)
- **Управление заказами**:
  - Просмотр списка заказов (для менеджеров и администраторов)
  - Добавление, редактирование и удаление заказов (для администраторов)

## Права доступа

- **Гость (guest)**: Только просмотр товаров
- **Клиент (client)**: Просмотр товаров
- **Менеджер (manager)**: Просмотр товаров и заказов, поиск и фильтрация
- **Администратор (admin)**: Полный доступ ко всем функциям

## Структура

- `desktop_app.py` - главный файл приложения
- `desktop/` - модули десктопного приложения:
  - `auth_view.py` - экран авторизации
  - `products_view.py` - экран управления товарами
  - `orders_view.py` - экран управления заказами

## Отличия от веб-версии

- Использует Flet для создания GUI вместо HTML шаблонов
- Работает как нативное десктопное приложение
- Не требует веб-сервера
- Использует те же сервисы и модели данных
