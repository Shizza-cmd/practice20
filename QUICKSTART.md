# Быстрый старт

## 1. Установка

```bash
pip install -r requirements.txt
python migrations/init_db.py
python create_image_placeholder.py
```

## 2. Запуск

```bash
python run.py
```

Откройте браузер: http://localhost:8000

## 3. Вход в систему

- **Администратор**: `admin` / `admin123`
- **Менеджер**: `manager` / `manager123`
- **Клиент**: `client` / `client123`

Или войдите как гость, нажав "Просмотр как гость" на странице входа.

## Возможности по ролям

### Гость
- Просмотр товаров (без фильтров)

### Клиент
- Просмотр товаров (без фильтров)

### Менеджер
- Просмотр товаров (с фильтрацией, поиском, сортировкой)
- Просмотр заказов

### Администратор
- Все возможности менеджера
- Добавление/редактирование/удаление товаров
- Добавление/редактирование/удаление заказов

## Особенности интерфейса

- **Товары со скидкой > 15%**: зеленый фон (#2E8B57)
- **Товары со скидкой**: перечеркнутая цена красным, новая цена черным
- **Товары без остатка**: голубой фон
- **Поиск в реальном времени**: автоматическая отправка формы при вводе
- **Фильтрация и сортировка**: работают совместно
